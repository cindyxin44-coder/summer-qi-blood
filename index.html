情
SEVEN EMOTIONS
七情内伤
七情是正常情志反应，但「过度」或「长期压抑」就会内伤脏腑、耗损气血。点击展开。
💡 情绪本是正常反应，压抑它才是根本问题。每种情志都有正向价值，失衡才需要调。
😄
喜 → 伤心
过喜则心气涣散
→ 失眠·多梦·心神不宁
▾
深层洞察：喜型人有时不是真开心——刻意维持高能量，强撑笑脸，内耗极大。典型案例（喜＋忧＋思叠加）：长期担当情绪、报喜不报忧，晚上用报复性熬夜麻痹自己，结果：失眠成常态·莫名心悸。中医诊断：心气不足，心神失养。

调理：内关穴按揉；嘶字诀睡前引导；最关键：放下强迫性高能量，允许示弱；莲子心茶清心火。
😡
怒 → 伤肝
大怒则肝气上逆；怒则气上
→ 头痛·面红·血压升高·胁肋疼痛
▾
怒值低（6分以下）的真实含义：不是平和，而是压抑。怒气憋在肝经路径上，具体体现为：子宫/乳腺/甲状腺结节、痛经血块、太阳穴偏头痛。

怒的正向价值：有力量·行动力强·有担当——压抑才是问题。

调理：嘘字诀每日早起6次；按揉太冲穴；4-7-8呼吸法；玫瑰花+佛手柑+陈皮茶。怒气后绝不宜进食。
🤔
思 → 伤脾
思虑过度则脾气郁结；思则气滞
→ 食欲差·腹胀·没精神·肌肉松软
▾
焦虑≠单一问题，需辨证脏腑：
· 焦虑来自脾胃：思的情志爆棚，思则气结
· 焦虑来自肾气不足：恐和惊的情志过剩
· 焦虑来自肺气防御过强：过于敏感，「烟感报警器全开」

调理：揉足三里+中脘；姜枣茶早起喝；每日20—30分钟有氧运动；减少信息摄入，每天给大脑留白。
😔
忧 → 伤肺
忧愁耗肺气
→ 干咳·气短·声音嘶哑
▾
深层洞察：夏季贪凉（空调+冷饮）会加重忧伤肺的损耗。形寒饮冷伤肺，体质弱者尤其易出现咳喘痰多。

调理：搭鹊桥生津养肺；按揉中府穴；夏季薄外套护腰肩；戒冰饮冷食；百合莲子粥润肺养心。
😢
悲 → 伤肺
悲损肺气，气不宣降
→ 呼吸无力·语声低微·容易落泪
▾
深层洞察：持续悲伤→肺气不降→气血运行无力→全身乏力。悲与忧常并见，持续的悲伤状态是气血消耗的无底洞。

调理：适当流泪（情绪排泄，不压抑）后需补肺气；深呼吸练习；太渊穴按揉；补肺食物：白萝卜·百合·银耳。
😨
恐 → 伤肾
恐则气下，肾气下陷
→ 面色苍白·头晕，严重可二便失禁
▾
深层洞察：长期焦虑恐慌→肾气下陷→腰膝酸软·夜尿频多·意志消沉·健忘耳鸣。强力久坐湿地亦可伤肾（负重劳损、寒湿侵腰）。

调理：艾灸关元·肾俞；晚泡脚+揉涌泉100下；减少高刺激信息；规律作息固护肾气；黑豆·黑芝麻·核桃食补。
😱
惊 → 伤心
惊吓扰乱心神
→ 心悸·失眠·心神不安
▾
深层洞察：容易受惊者，多为心脾两虚或肾气不足。惊与恐常并见（恐惊伤肾），长期处于惊恐状态会同时损耗心气和肾气。

调理：按揉神门穴（安心要穴）；嘶字诀睡前引导；减少突然的声音刺激；建立稳定的生活秩序感。
情志调养四字法则
通
疏通为先，不压抑。痛则不通，郁则成病。
化
用呼吸·运动将情绪能量转化排出。
养
培养安全感，减少信息焦虑，规律作息。
补
情志稳定后，配合饮食·穴位调理脏腑。
💡 为什么方法不管用
很多焦虑·失眠·消化问题，找错了脏腑「端点」，再多方法也是打地鼠。情绪是因，身体症状是果。只处理果而不回到因，就是补了半天没有用的根本原因。

养
PERSONALIZED · CARE
分人群调养
不同人群的气血特点与调养侧重不同，选择你的类型。
经期四阶段完整管理
经前7—10天：疏肝期
玫瑰花+佛手柑+陈皮茶；避免上火失眠；勿大补；按揉太冲·期门。
经期：温通保暖期
红糖水（经前3天始）；益母草炖鸡蛋；避免寒凉；经期本身进补无效。
经后：补血养阴黄金期
血海空虚，此时补最有效；大枣·桑葚·阿胶·猪肝；揉血海+三阴交。
平时：疏肝通络日常
嘘字诀每日6次；拍带脉；腹股沟推拿。怒气压抑→憋在子宫/乳腺→直接影响月经质量。
乳腺与激素问题
根源是肝郁。调情志+疏肝是根本；揉太冲·膻中·期门；规律运动（快走·八段锦）。经前疏肝才能持续改善。
久坐办公族（最普遍）
每小时起身5分钟做双手托天理三焦（6次）；揉足三里+委中缓解腰背；下班拍八虚排毒；久坐伤肉，脾主肌肉，需保持运动激活脾气。
压力型肝郁（怒气压抑多）
嘘字诀每日早起6次；拍腋窝极泉打通心气；4-7-8呼吸法情绪激动时用；玫瑰陈皮茶；减少表情管理的内耗。
肾虚型（脱发·腰膝酸·夜尿多）
晚泡脚+揉涌泉；艾灸关元+肾俞；早起姜枣茶；黑芝麻·核桃·黑豆食补；减少恐慌情绪（恐伤肾）。
夏季贪凉注意
戒冰饮冷食；空调房薄外套护腰肾；出汗后及时擦干避风；形寒饮冷伤肺是夏季男性最常见气血损耗。
总体原则：固本培元，动静结合
气血双虚为主。久卧伤气，久坐伤肉——每日保持适量活动（散步30分钟+坐式八段锦）。以维持第3—4层（睡眠良好·精力基本够用）为目标，争取手足温暖（第5层）。
脾胃弱者
揉腹+足三里；食物宜软烂温热；忌夏日冰饮；焦三仙茶助消化。
肾虚明显
晚泡脚；黑芝麻·核桃·黑豆；避免长期恐慌情绪；艾灸关元。
睡眠差
嘶字诀入睡；睡前揉神门+涌泉；亥时（21—23点）入睡。
心脑供血不足
拍八虚；搭鹊桥随时做；每日散步；深呼吸。
最大风险：久视伤血
每用眼1小时，闭目养神5分钟；揉睛明·四白·太阳穴；嘘字诀明目（肝开窍于目）；减少深夜用眼（久视+熬夜双重耗血）。
脾胃弱（厌食偏食）
捏脊（由下向上）；揉足三里；焦三仙茶消食化积；三餐规律，暴饮暴食=大饱伤脾，根源是没有安全感。
情绪管理
情绪敏感·抵触压力→引导用运动排泄（羽毛球·跑步·游泳）；嘘字诀疏肝。肝郁高发期从25岁开始，青少年期打好基础很关键。
夏季作息（最关键）
22点前入睡，让气血充分修复。面色差·眼神无光（第1—2层）→最快的办法就是改善睡眠，而不是进补。
孕期注意事项
⛔禁忌穴位：三阴交·合谷·至阴·昆仑等活血穴，可能引起宫缩
✅可做：搭鹊桥·嘶字诀；轻柔顺时针揉腹；双手托天（幅度减小，不踮脚）
💛情志调养最重要：母体情绪直接影响胎儿；思虑过重（思则气滞）对孕期伤害最大
🌿泡脚：可用艾叶代替花椒，水温38—40°C，不超过20分钟
产后调养
· 产后血虚：红枣+阿胶+桑葚食补；产后42天后可揉血海+三阴交
· 产后肝郁（情绪低落）：玫瑰花茶+轻柔揉太冲；家人情感支持与功法同等重要
· 哺乳期：禁花椒等辛热（影响乳汁）；泡脚改用艾叶；饮食以温补为主
